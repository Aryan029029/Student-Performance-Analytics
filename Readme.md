## Student Performance Analytics System

This is a Python-based system for managing and analyzing student data and academic performance. It allows teachers or administrators to handle student records, insert marks, generate reports, and visualize class performance.

## Technologies Used

1. Python 3.x

2. SQLite for storing student information and marks

3. Matplotlib for visualizing performance and generating graphs

4. Pandas for data analysis

5. Git and GitHub for version control

## Features
1. create and manage a student database

2. Insert student personal information and marks

3. View student records

4. Generate individual student reports

5. Analyze class performance with averages and topper information

6. Visualize data using graphs

## File Structure

01_create_database.py : Creates the database

02_insert_data.py : Inserts student details

03_view_data.py : View student information

04_marks_table.py : Creates marks table

05_insert_marks.py : Inserts marks

06_join_students&Marks.py : Joins student and marks tables

07_clean-marks.py : Cleans marks data

08_avg.py : Calculates average marks

09_student_report.py : Generates individual student reports

10_main_menu.py : Main menu interface

11_class_graph.py : Visualizes class performance

topper.py : Finds top-performing student
